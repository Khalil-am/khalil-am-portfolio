For years, enterprise business consulting has been weighed down by the same repetitive work: writing user manuals from scratch, assembling UAT test case workbooks, translating notification templates into bilingual sets, and replicating executive dashboards across environments. These are high-effort, low-creativity tasks that quietly consume days — sometimes weeks — of every engagement.

**BC Automations** was built to change that.

Live at [bc-automations.click](https://www.bc-automations.click), BC Automations is a growing library of **AI-powered automation templates** designed specifically for business consultants working with enterprise project management platforms — **PPlus, SPlus, and Diwan**.

---

## What BC Automations Actually Delivers

Every template inside BC Automations is a tested, production-ready automation that takes a real consulting deliverable and turns it into a structured, AI-driven workflow.

### **User Manual Generation**
Generate complete, production-grade user manuals from live systems or business requirement documents — across PPlus, SPlus, and Diwan. What used to take 2–3 weeks of writing, screenshot capture, and formatting now runs as a single automated pass.

### **UAT Test Case Workbooks**
Turn a configuration manual into a ready-to-execute UAT workbook. Test cases, preconditions, expected results, traceability — all generated, validated, and structured for immediate execution.

### **Bilingual Notification Templates**
Extract notification definitions directly from live systems and generate bilingual (Arabic + English) templates ready to be wired in — no copy-paste, no translation round-trips.

### **Dashboard Replication**
Replicate executive dashboards across PPlus instances using AI-driven configuration mapping. Instead of rebuilding dashboards manually in every environment, the automation understands the source and reconstructs it on the target.

### **Group Permissions Automation**
Convert permission matrices in Excel into API-driven configuration, eliminating the hundreds of manual clicks usually required to set up group-level permissions.

### **Migration Sheets for SPlus**
Inspect a live SPlus system and generate complete migration workbooks — capturing structure, relationships, and data shape in a format ready for migration execution.

## Why It Works: The MCP Layer Behind It

BC Automations is powered by a dedicated **Model Context Protocol (MCP) server** that exposes structured PPlus knowledge directly to Claude Code. Instead of forcing the model to guess how PPlus works, the MCP server provides 8 specialized tools backed by 44+ structured knowledge chunks — making every automation run on real, authoritative product context.

This is what separates BC Automations from "generic AI prompts":

- **Grounded in real product knowledge** — every automation uses verified PPlus/SPlus/Diwan context.
- **Deterministic where it matters** — configuration, APIs, and mappings are treated as structured data, not free-text guessing.
- **Reusable and versioned** — consultants share templates, not ad-hoc prompts.
